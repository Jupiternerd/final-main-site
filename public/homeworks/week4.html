Breakdown


The code is a little bit hard to read but after staring at it for a while, I was able to figure out some of what it does. The B var is the array for the text, the bottom P function animates using the X and Y properties every 999ms so around a 1s the X and Y variables have the .a and .b classes which are the two lines of text to be animated so they are being passed in differently with the B[0 or 1] Array access for different lines of text. The general p function is the main animation loop, it adjusts the initial rotation, font size, and the content. Some parts I don't understand are the "["一", "二", "三"][f(r() * 3)]" line and how it relates to the program. Even though the code is kind of hard to decipher, I think I can recreate it with a bit more time and playing around with it.
